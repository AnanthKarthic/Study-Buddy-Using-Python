# Study Buddy Using Python

### Objective

I created this Study Buddy application to help students prepare for finals by providing an interactive Python-based quiz system with immediate feedback.

### Skills Learned

- Python programming fundamentals
- User input handling and validation
- JSON-based data storage and file handling
- Data structures (lists and dictionaries)
- Designing interactive, quiz-based applications
  
### Tools Used

- Python for writing the application code
- JSON files for storing questions and answers
- GitHub for version control and project management
- Textbooks / study materials for content and question creation
